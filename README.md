# MultimodalCalorieRegressor

---

## Цель проекта

Создать нейросеть для предсказания калорийности блюд

---

## Выполненные задачи

1. Взят датасет с описанием блюд, проведена подготовка данных для обучения и тестирования
2. Была реализована мультимодальная модель, которая использует для определения калорийности три модальности: изображение блюда, вес, список ингредиентов
3. Модель была обучена на подготовленных данных. Ход обучения ориентировался на метрику MAE. В результате модель была протестирована на тестовом наборе данных, метрика MAE на наборе составила 46.66
